# Linked List Built From Scratch

A clean, low-level implementation of a Singly Linked List in C++ built using object-oriented classes and manual pointer manipulation, completely avoiding the standard `<list>` header file.


## Features

This implementation includes the following fundamental data manipulation operations:

* Insertion: `push_front`, `push_back`, and `insert` at any valid position.
* Deletion: `pop_front` and `pop_back` with manual heap memory cleanup.
* Utility: `search` for elements dynamically and `printLL` to visualize the list tracking.


## Compilation and Setup

Compile the file using any standard C++ compiler.
